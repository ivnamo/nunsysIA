# Archivo historico

Esta carpeta conserva documentacion, informes y prompts de fases anteriores.
No representa el flujo vigente de entrega.

Para evaluar el proyecto usa:

- `README.md`
- `docs/ARCHITECTURE.md`
- `docs/api.md`
- `docs/validation.md`
- `docs/VALIDACION_ENTREGA.md`

Los documentos marcados como `.legacy.md` y los informes bajo
`docs/archive/validation/` se mantienen solo como trazabilidad historica de
decisiones y validaciones previas.
